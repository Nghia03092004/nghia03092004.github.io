\section*{Overview}

Number theory in contest problems is less about memorizing isolated theorems and more about recognizing structure:
modular inverses, multiplicative functions, convolution-friendly moduli, or the shape of a factorization argument.

\section*{What These Notes Emphasize}

I prefer explanations that connect the algebra to the implementation. If a fact is only useful after it becomes a
linear sieve, a CRT merge, or an NTT butterfly, the note should say so directly.
