\section{Problem Statement}

A 123-number uses only digits {1,2,3}. Count/sum 123-numbers up to N with specific properties. Find F(111111111111222333) mod 123123123.

\section{Mathematical Analysis}

Digit DP with restricted digit set {1,2,3}. Track digits placed so far.

\section{Complexity Analysis}

O(D * states) where D = number of digits

\section{Answer}

\[
\boxed{57808202}
