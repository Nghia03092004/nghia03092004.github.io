\section{Problem Statement}
Define XOR-multiplication by carry-less polynomial multiplication over $\mathbb{F}_2$. A number $n>1$ is XOR-prime if it cannot be written as $a \otimes b$ with $a,b>1$. The task is to sum all XOR-primes up to $5 \times 10^6$.

\section{Archive Note}
This archived LaTeX note keeps a concise statement and the verified final answer for the problem. XOR-primes correspond to irreducible polynomials over $\mathbb{F}_2$.
