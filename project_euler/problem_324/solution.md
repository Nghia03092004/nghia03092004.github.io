# Problem 324: Building a Tower

## Problem Statement

Let $f(n)$ denote the number of ways to fill a $3 \times 3 \times n$ tower using $2 \times 1 \times 1$ bricks. Find $f(10^{10000}) \bmod (10^8 + 7)$.

## Mathematical Foundation

**Theorem 1 (Transfer matrix).** *Let $\mathcal{S} = \{0,1\}^9$ be the set of all bitmasks of the $3 \times 3$ cross-section. There exists a $512 \times 512$ matrix $T$ over $\mathbb{Z}$ such that*
$$f(n) = (T^n)_{0,0}$$
*where state $0$ represents the empty profile (no protruding bricks).*

**Proof.** We build the tower layer by layer. A *profile* $s \in \mathcal{S}$ records which of the 9 cells at the boundary between two consecutive layers are occupied by bricks protruding from below. Entry $T[s_1][s_2]$ counts the number of valid placements of $2 \times 1 \times 1$ bricks within a single layer such that all empty cells of profile $s_1$ are filled, and the cells protruding upward form profile $s_2$. The brick placements within a layer correspond to horizontal dominoes (within-row or within-column) and vertical bricks crossing the boundary. Since the starting and ending profiles are both empty (no protrusions at the bottom or top), $f(n) = (T^n)_{0,0}$. $\square$

**Lemma 1 (Linear recurrence).** *The sequence $f(n)$ satisfies a linear recurrence*
$$f(n) = c_1 f(n-1) + c_2 f(n-2) + \cdots + c_d f(n-d)$$
*of degree $d$ equal to the degree of the minimal polynomial of $T$ restricted to the cyclic subspace generated by $e_0$ (the unit vector for state 0).*

**Proof.** The Cayley--Hamilton theorem guarantees that $T$ satisfies its characteristic polynomial of degree at most 512. The sequence $(T^n)_{0,0} = e_0^T T^n e_0$ therefore satisfies the corresponding recurrence. The minimal polynomial of the sequence has degree at most $d \leq 512$. $\square$

**Theorem 2 (Polynomial exponentiation for large $n$).** *Let $m(x)$ be the minimal polynomial of the sequence $\{f(n)\}$ of degree $d$, working modulo the prime $p = 10^8 + 7$. Then*
$$f(n) \equiv \sum_{i=0}^{d-1} a_i \, f(i) \pmod{p}$$
*where $x^n \equiv \sum_{i=0}^{d-1} a_i x^i \pmod{m(x)}$ in $\mathbb{F}_p[x]$, computed via repeated squaring.*

**Proof.** Since $f(n) = e_0^T T^n e_0$ and $m(T)$ annihilates the relevant subspace, any polynomial identity $x^n \equiv r(x) \pmod{m(x)}$ translates directly to $T^n \equiv r(T)$ on that subspace, hence $f(n) = \sum a_i f(i)$. The polynomial $x^n \bmod m(x)$ is computed via $O(\log n)$ squarings in $\mathbb{F}_p[x] / (m(x))$. $\square$

**Lemma 2 (Berlekamp--Massey).** *Given $f(0), f(1), \ldots, f(2d-1)$, the Berlekamp--Massey algorithm recovers the minimal polynomial $m(x)$ of degree $d$ in $O(d^2)$ operations over $\mathbb{F}_p$.*

**Proof.** Standard result; see Massey (1969). $\square$

## Algorithm

```
function solve():
    p = 10^8 + 7

    // Step 1: Build transfer matrix T (512 x 512)
    T = build_transfer_matrix()

    // Step 2: Compute f(0), ..., f(2*512) by repeated matrix-vector multiplication
    v = unit_vector(0)   // e_0
    values = []
    for i = 0 to 1024:
        values.append(v[0] mod p)
        v = T * v mod p

    // Step 3: Find minimal polynomial via Berlekamp-Massey
    m(x) = berlekamp_massey(values, p)
    d = deg(m)

    // Step 4: Compute x^(10^10000) mod m(x) in F_p[x]
    //   Represent n = 10^10000 in binary (~33220 bits)
    //   Use repeated squaring of polynomials mod m(x)
    r(x) = poly_power_mod(x, 10^10000, m(x), p)

    // Step 5: Evaluate f(n) = sum of a_i * f(i)
    return sum(r.coeff(i) * values[i] for i in 0..d-1) mod p
```

## Complexity Analysis

- **Time:** $O(2^9 \cdot 2^9 \cdot E)$ for transfer matrix construction (where $E$ is the cost of enumerating tilings per state pair) + $O(d^2)$ for Berlekamp--Massey + $O(d^2 \log n)$ for polynomial exponentiation, where $d \leq 512$ and $\log_2(10^{10000}) \approx 33220$. Dominant term: $O(d^2 \log n)$.
- **Space:** $O(d^2)$ for the transfer matrix or $O(d)$ for the polynomial representation.
