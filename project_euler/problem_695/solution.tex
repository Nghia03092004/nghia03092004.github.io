\section{Problem Statement}

Three random points in unit square define 3 axis-aligned rectangles via their pairwise diagonals. Find expected area of the second-largest rectangle, to 10 d.p.

\section{Mathematical Analysis}

Order statistics of uniform random variables. E[area] = E[|x1-x2| * |y1-y2|] for each pair.

\section{Complexity Analysis}

O(1) via closed-form integration of order statistics

\section{Answer}

\[
\boxed{0.1017786859}
\]
