\section{Problem Statement}
Define $f(n)$ by replacing every prime factor of $n$ with $2$, so that $f(n)=2^{\Omega(n)}$. Let
\[
S(N)=\sum_{k=1}^{N} f(k).
\]
The task is to evaluate $S(10^{18})$.
