\section{Problem Statement}
Strictly increasing positive integers $a_0<\cdots<a_n$ with $|a_i^2-a_{{i-1}}a_{{i+1}}|\le 2$ for interior terms, $n\ge 4$. $G(N)$ = count with all terms $\le N$. Find $G(10^{{18}}) \bmod 10^9+7$.

\section{Analysis}
The condition $|a_i^2-a_{{i-1}}a_{{i+1}}|\le 2$ means the sequence is 'almost geometric'. For a geometric sequence $a_i = a_0 r^i$, $a_i^2 = a_{{i-1}}a_{{i+1}}$ exactly. The tolerance $\pm 2$ allows near-geometric integer sequences.
