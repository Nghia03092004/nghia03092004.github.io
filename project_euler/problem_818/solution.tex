\section{Problem Statement}

Cards are vectors in $\mathbb{F}_3^d$. A \textbf{SET} is a triple $(\mathbf{a}, \mathbf{b}, \mathbf{c})$ with $\mathbf{a} + \mathbf{b} + \mathbf{c} \equiv \mathbf{0} \pmod{3}$ coordinatewise. Count the number of SETs in a given collection.

\section{Linear Algebra over $\mathbb{F}_3$}

\begin{lemma}
Given any two distinct cards $\mathbf{a}, \mathbf{b} \in \mathbb{F}_3^d$, there is a unique third card $\mathbf{c} = -\mathbf{a} - \mathbf{b}$ completing the SET.
\end{lemma}

\begin{theorem}[Full Deck Formula]
The number of SETs in the complete deck $\mathbb{F}_3^d$ is:
\[
T(d) = \frac{3^{d-1}(3^d - 1)}{2}.
\]
\end{theorem}

\begin{proof}
Choose any two distinct cards: $3^d(3^d-1)$ ordered pairs. Each pair determines a unique SET. Each SET is counted $3 \cdot 2 = 6$ times (ordered pairs from an unordered triple). So $T = 3^d(3^d-1)/6$... This gives the wrong formula. Let us be more careful.

Choose an unordered pair $\{a, b\}$: $\binom{3^d}{2}$ ways. The third card $c = -a-b$ is always in the full deck. But we need $c \ne a$ and $c \ne b$. Since $a \ne b$ implies $c \ne a$ and $c \ne b$ (otherwise $a + a + b = 0 \implies 2a = -b \implies b = a$ in $\mathbb{F}_3$, contradiction). So every unordered pair gives a valid SET, and each SET is counted $\binom{3}{2} = 3$ times. Therefore:
\[
T(d) = \frac{1}{3}\binom{3^d}{2} = \frac{3^d(3^d-1)}{6} \cdot \frac{6}{3 \cdot 2} \cdot 3 = \frac{3^d(3^d-1)}{6}.
\]
Wait: $T = \binom{3^d}{2}/3 = 3^d(3^d-1)/6$. For $d=2$: $9 \cdot 8/6 = 12$. Correct.

Actually $3^{d-1}(3^d-1)/2 = 3^d(3^d-1)/6$ since $3^{d-1}/2 = 3^d/6$. So both forms agree.
\end{proof}

\begin{theorem}[Character Sum]
\[
|\{(\mathbf{a},\mathbf{b},\mathbf{c}) \in S^3 : \mathbf{a}+\mathbf{b}+\mathbf{c} = 0\}| = \frac{1}{3^d}\sum_{\mathbf{t} \in \mathbb{F}_3^d} \hat{S}(\mathbf{t})^3
\]
where $\hat{S}(\mathbf{t}) = \sum_{\mathbf{v} \in S} \omega^{\mathbf{t}\cdot\mathbf{v}}$, $\omega = e^{2\pi i/3}$.
\end{theorem}
