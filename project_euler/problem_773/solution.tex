\section{Problem Statement}
$S_k = \{2, 5\} \cup \{\text{first } k \text{ primes ending in 7}\}$. A $k$-Ruff number is not divisible by any element of $S_k$. $N_k = \prod_{s \in S_k} s$. $F(k)$ = sum of all $k$-Ruff numbers less than $N_k$ that end in digit 7. Given $F(3) = 76101452$. Find $F(97) \bmod 10^9+7$.
\section{Analysis}
\subsection*{Inclusion-Exclusion on Primes}

A $k$-Ruff number $m < N_k$ satisfies $\gcd(m, N_k) = 1$ (coprime to all elements of $S_k$). Since $N_k = \prod S_k$ and all elements of $S_k$ are prime, this is exactly the condition that $m$ is coprime to $N_k$.

\subsection*{Euler's Totient and Digit Constraint}

The sum of all integers $< N_k$ coprime to $N_k$ is $\frac{N_k \cdot \phi(N_k)}{2}$. But we additionally require the last digit to be 7.
