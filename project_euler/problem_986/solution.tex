\begin{theorem}
$R(n) = \sum_{\lambda \vdash n}\mathrm{rank}(\lambda) = 0$ for all $n$.
\end{theorem}
\begin{proof}
Conjugation $\lambda \mapsto \lambda'$ negates the rank: $\mathrm{rank}(\lambda') = -\mathrm{rank}(\lambda)$.
\end{proof}
