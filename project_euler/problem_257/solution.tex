\section{Problem Statement}

Count the number of triangles with integer sides $a \le b \le c$ and perimeter $p = a+b+c \le 10^8$ such that at least one angle bisector has integral length.

\section{Angle Bisector Formula}

The length of the angle bisector from vertex $A$ (between sides $b$ and $c$, opposite side $a$) is:
\[
t_a = \frac{1}{b+c}\sqrt{bc\left[(b+c)^2 - a^2\right]}
\]

Equivalently:
\[
t_a^2 = \frac{bc(b+c-a)(b+c+a)}{(b+c)^2}
\]

\section{Integrality Condition}

For $t_a \in \mathbb{Z}$, we require:
\[
(b+c)^2 \mid bc(b+c-a)(b+c+a)
\]
and the quotient must be a perfect square.

Let $s = b+c$. Then $a = p - s$ and the condition becomes:
\[
t_a^2 = \frac{bc(s-a)(s+a)}{s^2} = \frac{bc(2s-p) \cdot p}{s^2}
\]

must be a non-negative perfect square.

\section{Enumeration}

\subsection{Parametrization}

For a triangle with $a \le b \le c$:
\begin{itemize}
    \item Triangle inequality: $a + b > c$, equivalently $a > c - b$.
    \item Perimeter constraint: $a + b + c \le 10^8$.
\end{itemize}

\subsection{Algorithm}

We iterate over possible values of $s = b+c$ and $a$, checking:
\begin{enumerate}
    \item Triangle inequality: $a < s$ and $a \ge 1$.
    \item The expression $\frac{bc(s^2 - a^2)}{s^2}$ is a perfect square.
\end{enumerate}

Since $b + c = s$ and $b \le c$, we have $b \le s/2$ and $bc = b(s-b)$.

Using number-theoretic sieving and GCD-based simplification, the computation is feasible for $p \le 10^8$.
