The quadtree encoding allows us to describe a black and white image as a sequence of bits (0 and 1). Those sequences are to be read from left to right like this:

  • the first bit deals with the complete region;
  • "0" denotes a split:
    the current region is divided into sub-regions of dimension ,
    the next bits contains the description of the top left, top right, bottom left and bottom right sub-regions - in that order;
  • "10" indicates that the current region contains only black pixels;
  • "11" indicates that the current region contains only white pixels.
