The straightforward implementation recalculates the same chain lengths constantly.

## Useful observation

Many starting values quickly fall into sequences that have already been explored.

That makes memoization the obvious improvement:

```python
cache = {1: 1}

def length(n: int) -> int:
    if n in cache:
        return cache[n]
    nxt = n // 2 if n % 2 == 0 else 3 * n + 1
    cache[n] = 1 + length(nxt)
    return cache[n]
```
